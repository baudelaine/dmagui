SQL queries for labels and descriptions

SQL syntax
SELECT ... AS Table_Name, ... AS Table_Label FROM ... IN (?)
About parameter(s)
(?) stand for table(s) selected in 'Choose table(s)'.
e.g.
SELECT table AS Table_Name, label AS Table_Label FROM tablelabels where table IN (?) and lang = 'fr'

Everything in bold is mandatory.
e.g. MAXIMO
SELECT MAXIMO.MAXTABLE.TABLENAME AS Table_Name, MAXIMO.MAXOBJECT.DESCRIPTION AS Table_Label FROM MAXIMO.MAXTABLE join MAXIMO.MAXOBJECT on MAXIMO.MAXTABLE.TABLENAME = MAXIMO.MAXOBJECT.OBJECTNAME where MAXTABLE.TABLENAME IN (?)

About parameter(s)

e.g.

SQL syntax
SELECT ... AS Table_Name, ... AS Column_Name, ... AS Column_Label FROM ... = ? ... IN (?)
About parameter(s)
? stand for each table selected in 'Choose table(s)'.
(?) stand for all columns from each table(s) selected in 'Choose table(s)'.
e.g.
SELECT table AS Table_Name, column AS Column_Name, label AS Column_Label FROM columnlabels where table = ? and column IN (?) and lang = 'fr'

Everything in bold is mandatory.
e.g. MAXIMO
SELECT maxattribute.objectname AS Table_Name, maxattribute.attributeName AS Column_Name, maxattribute.title AS Column_Label FROM maxattribute where maxattribute.objectname = ? and maxattribute.attributeName IN (?)
